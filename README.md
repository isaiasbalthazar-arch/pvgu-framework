## About the Project

The **Universal Geometric Framework (UGF/PVGU)** is an independent scientific research program initiated in 2011. It investigates **geometric persistence**, **effective spacetime structure**, and **multiscale coherence** across different physical domains.

This repository (`pvgu-framework`) serves as the **central scientific infrastructure** of the project (PVGU-RCUR), aiming to organize, document, version, and make reproducible all research outputs developed over the years.

---

## Current Status

The project is currently in the **consolidation phase**. The main goals at this stage are:

- Unifying dispersed experiments and analyses into a coherent structure
- Establishing standardized metadata and taxonomies
- Creating a reproducible and well-documented scientific repository
- Defining clear epistemological and methodological foundations

> **Note:** This repository is under active development. Many files and folders are still being organized and migrated from the original research archive.

---

## Key Objectives

- Consolidate all experiments, results, and documentation into a single, organized structure
- Implement standardized metadata and a master catalog
- Enable cross-scale validation and reproducibility
- Provide clear epistemological grounding (PNO, ONTI, H5, etc.)
- Serve as a permanent, citable scientific resource
